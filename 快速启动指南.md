# Excalidraw AI 后端服务 - 快速启动指南

## 🚀 一键启动

### Windows 用户
双击运行 `start-backend.bat` 文件

### Mac/Linux 用户
在终端运行：
```bash
chmod +x start-backend.sh
./start-backend.sh
```

## 📋 启动步骤说明

启动脚本会自动执行以下步骤：

1. ✅ **检查环境** - 验证 .env 文件是否存在
2. 🔨 **编译代码** - 将 TypeScript 编译为 JavaScript
3. 🚀 **启动服务** - 在端口 3016 启动后端服务

## 🌐 服务地址

启动成功后，以下地址可用：

- **API 端点**: http://localhost:3016/v1/ai/text-to-diagram/chat-streaming
- **健康检查**: http://localhost:3016/health

## 🛑 停止服务

在运行窗口按 `Ctrl + C` 即可停止服务

## ⚠️ 常见问题

### 端口被占用
如果看到 "Port 3016 is already in use" 错误：
```bash
# Windows
netstat -ano | findstr :3016
taskkill /F /PID <进程ID>

# Mac/Linux
lsof -ti:3016 | xargs kill -9
```

### 找不到 .env 文件
确保在 `excalidraw-ai-backend` 目录下有 `.env` 文件，包含：
```
ZHIPUAI_API_KEY=你的API密钥
ZHIPUAI_MODEL=glm-4.7
PORT=3016
NODE_ENV=development
RATE_LIMIT_LIMIT=100
RATE_LIMIT_WINDOW_MS=86400000
```

### 编译失败
确保已安装依赖：
```bash
cd excalidraw-ai-backend
npm install
```

## 📝 手动启动

如果不想使用启动脚本，可以手动执行：

```bash
cd excalidraw-ai-backend
npm run build  # 编译
npm start      # 启动
```

## 🎯 测试服务

启动后，在浏览器访问 Excalidraw，打开 AI 助手，选择"思维导图"标签，输入主题并生成！
